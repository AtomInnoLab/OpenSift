# Interactive Demo

Experience the difference AI-powered search verification makes. The demo below shows a side-by-side comparison: **traditional search** vs. **OpenSift-enhanced search** on the same query and results.

!!! tip "How to use"
    1. Select one of the pre-loaded queries
    2. Click **Run Comparison**
    3. Watch the AI pipeline process results in real-time
    4. Expand **"Show AI assessment"** on any result to see criteria-level reasoning
